#  Rediseño Web IPS Clinaid

Este proyecto es una propuesta de **rediseño de la página web de la IPS Clinaid**, desarrollado con el objetivo de mejorar la experiencia del usuario mediante la aplicación de principios de **Diseño de Interfaces (UI)** y **Experiencia de Usuario (UX)**.  
El enfoque del proyecto se centra en optimizar la **usabilidad**, **accesibilidad**, **rendimiento** y **estética visual** del sitio.

---

##  Objetivos del Proyecto

- Mejorar la **navegación** y estructura general del sitio.
- Aumentar la **accesibilidad**, asegurando que el contenido sea apto para todo tipo de usuarios.
- Optimizar el **rendimiento** mediante buenas prácticas en HTML, CSS y JavaScript.
- Modernizar la **estética** manteniendo coherencia con la imagen corporativa de la IPS Clinaid.
- Crear una interfaz más intuitiva siguiendo referentes y metodologías **UX**.

---

##  Características Principales

- Rediseño visual basado en un enfoque minimalista, limpio y profesional.
- Estructura reorganizada para mejorar el flujo de información.
- Tipografías, colores y componentes visuales alineados con Clinaid.
- Secciones más claras para servicios, citas, contacto y perfil institucional.
- Mejora en jerarquía y legibilidad del contenido.
- Sitio completamente **responsive** para distintos dispositivos.

---

##  Tecnologías Utilizadas

- **HTML5**  
- **CSS3**  
- **JavaScript**
- **Git & GitHub**
- Herramientas de prototipado UX/UI (Figma / Adobe XD) *(opcional según tu proyecto)*

---

##  Metodología UX Aplicada

- Análisis de la página original  
- Identificación de problemas de usabilidad  
- Wireframes y estructura mejorada  
- Propuesta gráfica basada en principios UX (contraste, jerarquía, consistencia, accesibilidad)  
- Desarrollo del sitio optimizado
